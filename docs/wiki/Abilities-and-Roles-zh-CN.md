# 能力与身份

> **Wiki 页面版本：** zh-CN 1.0.0 · **英文源版本：** EN 1.0.0 · **产品基线：** v0.1.3 · **更新日期：** 2026-07-18
> **English:** [Abilities and Roles](Abilities-and-Roles) · **同步状态：** 已同步至 EN 1.0.0

能力描述广义潜力，身份描述习得的社会与地理经验。两者都不是僵硬职业。目录驱动的计算
允许内容包添加稳定能力/身份 ID，不需要修改中心 switch。

## 八项基础能力

| 能力              | 含义                       | 示例用途                             |
| ----------------- | -------------------------- | ------------------------------------ |
| strength 力量     | 推拉、举重、重型操控、冲击 | 负重、武器需求、强制位移             |
| agility 敏捷      | 平衡、精细动作、闪避、准确 | 操控、恢复、攀爬、武器处理           |
| endurance 耐力    | 持续活动与环境耐受         | 生命、体力、冷热耐受                 |
| intelligence 智力 | 分析、工程、记忆、复杂技巧 | 制作、研究、复杂动作                 |
| perception 感知   | 侦测、导航、远距观察       | 视野、陷阱、追踪、瞄准               |
| willpower 意志    | 专注、抗恐惧、仪式耐久     | 专注、决心、抗打断                   |
| charisma 魅力     | 说服、领导、社交判断       | 谈判、士气、声誉                     |
| luck 幸运         | 狭义不确定性修正           | 稀有结果和同值判定，不能替代其他能力 |

## 派生资源

公式为 `value = round(base + Σ ability[id] × weight[id])`。首批资源：生命、体力、
专注、韧性、负重、决心。这里只计算容量；消耗、伤害、恢复和状态属于后续可变状态。

## 背景身份

| 身份       | 地理关系           | 初始优势             |
| ---------- | ------------------ | -------------------- |
| 寻路者     | 混合边疆           | 导航、露营、观察     |
| 河川守望   | 河流、三角洲、湿地 | 船、绳索、洪水判断   |
| 关隘卫士   | 山地与道路         | 耐力、长柄、路线防御 |
| 档案学者   | 城镇与遗迹         | 文书、语言、分析     |
| 行医者     | 聚落与远行         | 药物、诊断、暴露处理 |
| 工艺匠人   | 矿山、森林、城镇   | 修理、材料、工具     |
| 商队信使   | 道路与沙漠         | 物流、骑乘、谈判     |
| 仪式修习者 | 神圣与危险地点     | 专注、决心、本地仪式 |

身份提供能力倾向、熟练标签、地形亲和和起始物品标签，未来可影响社会识别和导师，但不
阻止学习其他领域。新能力需要 ID、边界、本地化、校验和显式公式权重；新身份必须说明
地理、熟练、物品与文化背景，避免把身份做成纯战斗套装或族群刻板印象。
