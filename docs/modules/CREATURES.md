# Creature Module

**Status:** deferred plugin.

## Owns

Species definitions, habitat suitability, spawn populations, behavior policies,
observation records, and optional collection mechanics.

## First Vertical Slice

Generate one original species from configuration, spawn it only in suitable habitat,
let the explorer observe it, and store the discovery locally.

## Dependencies and Tests

Depends on map ecology, weather, time, and characters. Test habitat filters, population
bounds, seeded spawning, save restoration, and protected-IP content checks.
